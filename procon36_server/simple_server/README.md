# procon36 競技サーバー ユーザーマニュアル

## 起動方法

```bash
./procon36_server -listen :3000 -config match.json
```

### -listen

サーバーが接続を受け付けるアドレスを指定します。
接続されているどのネットワークからでも接続できるようにしたい場合は `:3000` のように設定してください。
実行しているマシン以外から接続できなくするには `127.0.0.1:3000` や `localhost:3000` のように設定してください。

### -config

試合の設定ファイルのパスを指定します。
指定しない場合は別途試合作成APIを利用して試合を作成する必要があります。
試合作成APIについては別途api_spec.htmlを参照してください。

## 試合の設定ファイル

試合の設定ファイルは下記の形式の JSON で記述します。
各フィールドの詳細については試合作成APIのドキュメントを参考にしてください
※試合作成APIのリクエストとはフィールド名など一部異なりますので注意してください。

設定ファイルの例
```json
{
  "match": {
    "players": [
      {"token": "player1", "name": "高専1"},
      {"token": "player2", "name": "高専2"}
    ],
    "problem": {
      "field": [
        [6, 3, 4, 0],
        [1, 5, 3, 5],
        [2, 7, 0, 6],
        [1, 2, 7, 4]
      ]
    },
    "duration": 300
  },
  "start": 10
}
```
